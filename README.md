# Airbnb Clone

nbnb is a full-stack web application inspired by Airbnb.

Features

    Custom User Authentication
  
    Search rooms with filter results
  
    Individual room pages
  
    User reviews on Rooms
  
    Book rooms
  
    Conversations
    
    User Favorite List
  
Technologies Used

    FrontEnd: Django, HTML, Tailwind CSS
   
    Backend: Python
  
    Database: dbsqlite
